sinar.pardocs Installation
--------------------------

To install sinar.pardocs using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``sinar.pardocs`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        sinar.pardocs

* Re-run buildout, e.g. with:

    $ ./bin/buildout
